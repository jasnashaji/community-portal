## ✅ Features

- View upcoming events
- Filter events by category or location
- Register for events with a simple form
- Geolocation support to show nearby events
- Character count feedback
- Responsive layout with modern CSS

## 💻 Technologies Used

- HTML5
- CSS3 (Modular architecture)
- JavaScript (Vanilla)

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `index.html` in a web browser.
3. Explore the navigation and functionality.

## 📝 Notes

- All scripts are handled via `script.js`.
- CSS is modularized and linked in the `<head>` of each HTML file.
- Preferences are stored using `localStorage`.
